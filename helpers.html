(static) insert(table, obj, optionsopt) → {string}

Generates a complete INSERT query from an object, using its properties as insert values.

Parameters:
Name Type Attributes Description
table String

Destination table name.

Passing in anything other than a non-empty string will throw TypeError = Parameter 'table' must be a non-empty text string.
obj Object

Object with properties for insert values.

Passing in anything other than a non-null object will throw TypeError = Parameter 'obj' must be a non-null object.
options helpers.propertyOptions <optional>

An object with optional parameters.

Passing in anything other than a non-null object will be ignored.

Examples
// Default usage

var obj = {
   one: 123,
   two: 'test'
};

var query = pgp.helpers.insert('myTable', obj);
//=> INSERT INTO "myTable"("one","two") VALUES(123,'test')
// Advanced usage, with `exclude` and `defaults`

var obj = {
    zero: 0,
    one: 1,
    two: undefined,
    // `three` is missing
    four: true
};

var query = pgp.helpers.insert('myTable', obj, {
    exclude: 'zero', // exclude property `zero`
    defaults: {
        one: 123, // use `one` = 123, if missing;
        three: 555, // use `three` = 555, if missing;
        two: function (value) {
            // set `two` = `second`, if it is `undefined`,
            // or else keep the current value:
            return value === undefined ? 'second' : value;
        },
        four: function (value) {
            // if `one` is equal 1, set `four` to `false`,
            // or else keep the current value:
            return this.one === 1 ? false : value;
        }
    }
});
//=> INSERT INTO "myTable"("one","two","four","three") VALUES(1,'second',false,555)

Type Definitions

propertyOptions

Set of rules used when formatting queries.

Properties:
Name Type Attributes Default Description
include String | Array <optional>

Either a single string with a property name, or an array of strings.

By default, all properties are used as insert values. But if include is set to a string or an array, then only the specified properties will be used, ignoring the rest.
exclude String | Array <optional>

Either a single string or an array of strings - names of properties to be excluded.

Excluding a property name that's present on the include list will throw TypeError = Property 'propName' is included and excluded at the same time.

Excluding a property name that's present within the defaults option will throw TypeError = Cannot exclude property 'propName', as it has a default value.
defaults Object <optional>

An object with required properties set to their default values. Such values will be used whenever the corresponding property doesn't exist within obj.

Setting a property to a function is treated as an override for the value, and the call into defaults.propName(value) is expected to return the new value for the property. The function is called with parameter value - the original value within obj, and with this context set to obj, i.e. value = this.propName.
inherit Boolean <optional>
 false

Triggers the use of inherited properties within obj (in addition to its own).

By default, only the object's own properties are enumerated for insert values.
